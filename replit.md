# FCC Infosec Challenges

## Overview
FreeCodeCamp Information Security and Quality Assurance challenges boilerplate. A Node.js Express application with Helmet security middleware.

## Project Architecture
- **Framework**: Express.js
- **Security**: Helmet middleware for HTTP security headers
- **Port**: 5000 (bound to 0.0.0.0 for Replit compatibility)

## File Structure
- `myApp.js` - Main application entry point, configures Helmet and starts server
- `server.js` - API routes and middleware (DO NOT EDIT - used for verification)
- `views/index.html` - Main HTML page
- `public/style.css` - Styles

## Running the App
```bash
npm start
```
